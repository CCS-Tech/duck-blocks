Let's Create our first Web Page

First, let's name the page's file

This will be the label put onto the HTML file we're about to create. For example, if you enter "home" as your page name, the file "home.html" will be created containing the data we'll entering later.

Now, we'll create the head of the page

The head of the web page is not visible when to page loads in a web browser. Rather, it contains information that pertains to settings of the page and the location of other files and scripts that will be used throughout the page itself. It also contains the title of the page, which will be visible on the tab for your page on the top of your browser.

The character set is the list of characters (letters, numbers, and specials) that will be present on your web page. If your website will be in English, UTF-8 or ASCII are the most common and either should be fine.
